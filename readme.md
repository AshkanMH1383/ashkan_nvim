i => Insert Mode

x,X => Remove Char

:wq => Save and Exit

r => Replace

o,O => Add New Line

u => Undo

dw => Delete a Word

dd => Delete a Line

J => Merge Lines

10G, 1G, ... , nG => Go to Line n 

y => Copy

yy => Copy a Line

yw => Copy a Word

P => Paste

/ => Search

$ => End of a Line
